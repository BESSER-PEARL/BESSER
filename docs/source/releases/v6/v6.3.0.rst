Version 6.3.0
=============

New Features
-------------

**BESSER Action Language Integration**

- Added full **BESSER Action Language (BAL)** grammar support
- Implemented a new **BAL editor** with syntax highlighting.
- Added BAL User documentation (Language overview and standard library description)

**Web App Generator Enhancements**

- Added support for BAL Methods
- Added modal for return of functions in the web app generator, allowing users to easily view and utilize function outputs.

**Backend Generator Enhancements**
- Added support for BAL Methods

Fixes
-----

- Fix #413: **Graphical UI editor** — "Instance Source" dropdown in button's Run Method action now properly displays all available table sources. Fixed component type detection to work with both manually placed and auto-generated tables.
- Fix #425: **Web app generator** — Method button modal now renders date, datetime, and time parameters with the correct HTML input types instead of plain text fields.
- Fix #22: **BUML metamodel** — Add type checks in structural.py to ensure model integrity and prevent invalid configurations.
- Fix #406: Added a Troubleshooting section to the documentation to help users resolve common issues and errors when using BESSER.
